Expected format

A JSON array of objects (a single object is also accepted):

[
  {
    "name": "Senior Backend Engineer",
    "company": "Acme Corp",
    "url": "https://example.com/jobs/123",
    "salary": "€90,000 - €100,000/year",
    "description": "Full job description text...",
    "keywords": ["Python", "Flask", "MongoDB"],
    "state": "open"
  }
]

state is optional (defaults to open). A closed opportunity closes the matching job if one exists, otherwise it is imported as a closed record for keyword/statistics.
